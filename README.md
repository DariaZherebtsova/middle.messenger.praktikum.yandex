## Установка

- `npm install` — установка
- `npm run build:dev` — development-сборка
- `npm run webpack:serve` - запуск webpack dev-сервера
- `npm run serve` — запуск express dev-сервера

#### Тесты

- `npm run test`

#### Линтинг

- `npm run eslint`
- `npm run stylelint`
